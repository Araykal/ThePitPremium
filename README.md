#ThePitPremium

## Project Overview

A Minecraft mini-game plugin, based on the Spigot 1.8.8 implementation.

## Dependency

- **[critical]** Need Java 11, Spigot API 1.8.8
- **[critical]** Data based on MongoDB, Redis
- **[critical]** The title requires LuckPerms
